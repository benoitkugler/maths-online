Politique de  confidentialité

Dans le souci de respecter le Règlement Général de Protection des Données, nous n'échangeons aucune information personnelle avec un tiers.

L'application élève Isyro ne collecte aucunes données personnelles.
Les données collectées par la plateforme enseignant (nom, prénom et date de naissance) sont utilisées exclusivement pour un usage interne à l'application.

Vous pouvez nous demander l’accès aux données personnelles vous concernant soit :
  • pour vous en informer,
  • pour rectifier des informations inexactes,
  • pour en demander l’effacement (droit à l’oubli),
  • pour en limiter les usages.
